# ITI Assignments (9-Months)

This repository contains my **assignments and coursework completed during the ITI 9-Months training program**. Each folder represents a specific course or topic, and includes practical tasks, labs, and exercises.

## Repository Structure

### Core Programming & Object-Oriented Principles
* **Java/**
  Java programming assignments covering core concepts, OOP principles, and problem-solving exercises.
* **OOP-C++/**
  C++ assignments with a focus on Object-Oriented Programming concepts, syntax, and best practices.
* **Kotlin/**
  Core Kotlin programming concepts, syntax, and exercises.
* **Dart/**
  Dart programming assignments focusing on language fundamentals and object-oriented principles.

### Data Structures & Architecture
* **Data-Structures/**
  Implementations and exercises for data structures and algorithms such as arrays, linked lists, stacks, queues, and related problems.
* **Design-Patterns/**
  Implementations of common software design patterns to solve recurring architectural problems and build scalable code.

### Mobile Development
* **Android-Kotlin/**
  Android development assignments and lab work implemented using Kotlin.
* **Mobile-Java/**
  Mobile app development assignments and projects built using Java.
* **Flutter/**
  Cross-platform mobile development assignments and projects using the Flutter framework (includes projects like `crypto_trend`).
* **Material-Design/**
  Tasks focused on the application of Material Design principles, UI/UX components, and styling.
* **Coroutines-kotlin/**
  Assignments focused on asynchronous programming and concurrency using Kotlin Coroutines.
* **RXJava/**
  Exercises and implementations utilizing reactive programming concepts with RxJava.

### Web Development & Data Formats
* **Web/**
  Web development assignments using **HTML, CSS, and JavaScript**, focusing on building structured and responsive web pages.
* **Web-RestFul/**
  Web development assignments focused on building and interacting with RESTful APIs.
* **XML & Json/**
  Assignments related to **XML and JSON**, including data representation, parsing, and handling structured data formats.

### Testing
* **Testing/**
  Assignments related to software testing concepts, tools, and practical testing exercises.
* **Android-testing/**
  Specific assignments and implementations focused on testing Android applications.
